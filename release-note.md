SteaMidra v6.3.4

What's new:

* Home page hint banner is now collapsible. The SteamStub / EAC / Content Still Encrypted info can be hidden or shown with one click.
* DLC Unlockers card moved to the bottom with Quick Tools. DLC Check is now the main DLC tool, while CreamAPI and SmokeAPI unlockers are still available when needed.
* “Let Updates” button renamed to “Auto Update”.
* Crack Files picker now shows the Build ID from crackfiles.json next to each game, making it easier to match fixes with your installed version.
* Bulk import cancel now properly hides the progress bar.
* GMRC HTTPS mirror fallback decrypt step fixed.
* Cloud Saves GDrive status now shows the real connected/disconnected state.
* Provider depot key cache now clears correctly after provider refreshes, so updates apply without restarting.
* Update checker now returns depot tokens correctly alongside games.
* Settings and Steam libraryfolders.vdf writes are now atomic, reducing the chance of corrupted files if the app crashes during saving.
* Store tab now blocks duplicate Search clicks, preventing duplicate search threads.
* Lua generator performance improved by loading provider depot keys once per render instead of once per depot.
* Several internal bug fixes and cleanup, including bridge queued-call replay, LumaCore add_ids logging, DLC unlocker permission validation, skeleton card CSS cleanup, and datetime cache updates.

Full detailed changelog is in CHANGELOG.md
